# Mindkind Website

The official website for [Mindkind](https://mindkind.ai) - The Cognitive Community.

## About

Mindkind explores the emerging community of human and synthetic minds coexisting. This site accompanies the book *Mindkind: The Cognitive Community* published by Kwalia Books.

## Deployment

Hosted on GitHub Pages with custom domain `mindkind.ai`.
